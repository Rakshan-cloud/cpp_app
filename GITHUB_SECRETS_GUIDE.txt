============================================================
  GITHUB SECRETS TO ADD FOR CI/CD DEPLOYMENT
============================================================

Go to: https://github.com/Rakshan-cloud/cpp_app/settings/secrets/actions

Click "New repository secret" for each of the following:

------------------------------------------------------------
1. AWS_ACCESS_KEY_ID
------------------------------------------------------------
   Value: Your AWS Access Key ID
   Purpose: AWS authentication for deploying backend (EB) and frontend (S3)

------------------------------------------------------------
2. AWS_SECRET_ACCESS_KEY
------------------------------------------------------------
   Value: Your AWS Secret Access Key
   Purpose: AWS authentication (paired with Access Key ID)

------------------------------------------------------------
3. API_URL
------------------------------------------------------------
   Value: http://rakshan-prod.eba-7mxtcx4w.ap-southeast-2.elasticbeanstalk.com/api
   Purpose: Frontend needs this to call the backend API
   Note: This is the Elastic Beanstalk URL + /api

============================================================
  DEPLOYMENT ORDER
============================================================

Step 1: Add AWS_ACCESS_KEY_ID and AWS_SECRET_ACCESS_KEY to GitHub Secrets
Step 2: Add API_URL to GitHub Secrets
Step 3: Push code to main branch
Step 4: GitHub Actions will: test library → deploy backend → deploy frontend
